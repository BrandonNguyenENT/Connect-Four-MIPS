# Connect-Four-MIPS
Connect Four Game -  Video-Game Programming (MIPS Assembly)

## Purpose

This repository contains all important files related to the project made in this course. This project is a standard Connect 4 game that consists of two player turn-based I/O, a random number generator (RNG) system, conditional branches (if-else), nested loops (while(), for(), do-while()), usage of a 2D array, 3 levels of subroutine calls, usage of the stack to store at least the $ra (return address) register to enable 3 levels of nested subroutine calls, sound, and more.

## Project Requirements

- Conditional branches, equivalent to C if-else

- Nested Loops equivalent to C while(), for() or do-while()

- Use of a 2-D array

- At least 3 levels of subroutine call: main jal to child function, child jal to grandchild function

- Use of the stack to store at least the $ra (return address) register to enable >=3 levels of nested subroutine calls

## Contents

- Project Final Report: Proper documentation of project including description, purpose, images, C draft code, MIPS Assembly finaL code, and answered questions.
